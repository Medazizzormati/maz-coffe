## 🚀 Fonctionnalités

### 🌐 Pages du site
- **Accueil** : Présentation, slideshow, valeurs
- **Menu** : Catalogue des produits avec filtres
- **Contact** : Formulaire et informations de contact

### 🛒 Système de panier avancé
- Panier coulissant latéral
- Ajout/retrait de produits
- Calcul automatique du total
- Persistance des données (localStorage)
- Badge dynamique sur l'icône panier

### ✨ Features techniques
- **Design responsive** (Mobile, Tablette, Desktop)
- **Animations fluides** (transitions CSS)
- **Slideshow automatique** avec navigation manuelle
- **Filtrage avancé** des produits (catégorie, prix, tri)
- **Formulaire de contact** avec validation
- **Navigation hamburger** pour mobile

## 🛠️ Technologies utilisées

- **HTML5** - Structure sémantique
- **CSS3** - Animations, Flexbox, Grid, Variables CSS
- **JavaScript (ES6)** - Interactivité
- **Font Awesome** - Icônes
- **Google Fonts** - Typographie
- **Unsplash** - Images de qualité
